# Запуск рекламы на botisk.ru — Яндекс.Директ

## 1. UTM-разметка ссылок

Базовый URL: `https://botisk.ru/`

**Шаблон с автоподстановкой параметров Директа:**
```
https://botisk.ru/?utm_source=yandex&utm_medium=cpc&utm_campaign={campaign_name|campaign_name}&utm_content={ad_id}&utm_term={keyword}&yclid={yclid}
```

Метрика 109293181 распарсит автоматически. В Директе вставлять как «Ссылка» в каждом объявлении. `{campaign_name}`, `{ad_id}`, `{keyword}` и `{yclid}` — это plead-ы Директа, он подставит реальные значения.

## 2. Сегменты (используйте audience'ы бота)

В боте уже найдены 4 рабочих аудитории:

| Сегмент | Кому показывать | Креатив (видео) |
|---|---|---|
| **Память** | Тем, кто потерял близких / хочет «оживить» старое фото | `videos/warm_memory.mp4` |
| **Детство** | Родители младенцев/детей. «Оживите детство ребёнка» | `videos/childhood.mp4` |
| **Питомцы** | Любители собак/кошек, потерявшие питомца | `videos/pet.mp4` |
| **Love story** | Молодожёны, годовщины | `videos/love_story.mp4` |

**Совет:** каждый сегмент — отдельная кампания в Директе с разным `utm_campaign` (например: `memory`, `babies`, `pets`, `love`). Так в Метрике увидите по каждому сегменту: CPC, конверсию в free-генерацию, конверсию в оплату.

## 3. Цели в Яндекс.Метрика (создать вручную в UI)

[metrika.yandex.ru](https://metrika.yandex.ru) → счётчик 109293181 → **Цели** → создать **JavaScript-событие** для каждой из 9:

| Идентификатор | Когда срабатывает | Важность |
|---|---|---|
| `PHOTO_UPLOADED` | Юзер загрузил фото | 🟡 средняя |
| `GENERATION_STARTED` | Нажал «Создать видео», валидация прошла | 🟢 высокая (микро-конверсия) |
| `FREE_GEN_COMPLETED` | Бесплатное видео готово | 🟢 высокая |
| `PAID_GEN_COMPLETED` | Платное видео готово | 🟢 высокая |
| `GEN_FAILED` | PiAPI вернул ошибку | 🔴 алерт |
| `PAYMENT_OPEN` | Открыл модалку оплаты | 🟢 высокая |
| `PAYMENT_REDIRECT` | Ушёл на ЮKassa | 🟢 высокая |
| `PAYMENT_SUCCESS` | Вернулся с успешной оплатой | 🟢 **главная конверсия** |
| `VIDEO_DOWNLOAD` | Скачал готовое видео | 🟡 средняя |

В Директе → Стратегия → можно поставить **оплату за конверсию** на `PAYMENT_SUCCESS` (заработает после 10–20 конверсий).

## 4. Воронка, которую увидите в Метрике

```
Визит → PHOTO_UPLOADED → GENERATION_STARTED → FREE_GEN_COMPLETED → PAYMENT_OPEN → PAYMENT_REDIRECT → PAYMENT_SUCCESS
 100%       ~30%              ~25%                    ~22%               ~5-8%           ~3-5%            ~2-4%
```

Цифры — типичные для подобных AI-сервисов. Хуже — оптимизируем фотку шаг (плохая загрузка, формулировка); лучше — масштабируем бюджет.

## 5. Минимальный тестовый бюджет

- **3 000–5 000 ₽ на 5–7 дней** на 1 кампанию (1 сегмент)
- CPC обычно 8–25 ₽ в РСЯ, 20–60 ₽ на Поиске
- Цель — собрать первые 50–100 конверсий в free-генерацию → калибровка
- После — масштабируем сегмент с лучшей экономикой

## 6. Что мониторить ежедневно (первая неделя)

1. **Дашборд** https://n8n.24isk.ru/op/ — общий статус workflow и БД
2. **Метрика** — воронка по дням
3. **PiAPI баланс** — https://piapi.ai/dashboard (сейчас $55, хватит на ~150–300 free-генераций при 5-секундных видео)
4. **БД `web_users`** — сколько новых, сколько с paid_credits

Раз в день psql:
```sql
SELECT
    DATE(first_seen) AS day,
    COUNT(*) AS new_users,
    COUNT(*) FILTER (WHERE free_used) AS used_free,
    COUNT(*) FILTER (WHERE paid_credits > 0) AS paid,
    SUM(paid_credits) AS total_paid_credits,
    SUM(total_generated) AS total_gens
FROM web_users
WHERE first_seen > NOW() - INTERVAL '7 days'
GROUP BY 1 ORDER BY 1 DESC;
```
