## Два пути

### Путь A: через Yandex.Direct API (быстрее, нужен токен)

Пользователь должен выдать OAuth-токен с правами на Direct и Метрику.
См. раздел «Получение OAuth» ниже.

После того как `YANDEX_OAUTH_TOKEN` появится в `.env` — Claude может:

1. Создать кампании через `POST https://api.direct.yandex.com/json/v5/campaigns`
2. Создать группы объявлений (AdGroups) через `POST .../adgroups`
3. Создать объявления (Ads) через `POST .../ads` (текстово-графические + видео)
4. Загрузить ключевые слова через `POST .../keywords`
5. Подключить Метрика-цели в стратегии через `OptimizeConversionsScheme`

**Не запускать модерацию автоматически** — финальная проверка кампаний
за пользователем. Claude собирает каркас, пользователь смотрит и нажимает
«Запустить» в UI Директа.

### Путь B: через UI Директа (медленнее, без токена)

Claude пишет пошаговую инструкцию что кликать, пользователь делает руками.
Используется если OAuth не получили.

## Структура кампаний (4 шт.)

Все кампании — на botisk.ru, регионы: Россия + СНГ (Москва, СПб, регионы
по эффективности; СНГ — мониторим, выключаем если CPC выше окупаемости).

- **Боль:** оживить старое фото близкого человека, бабушки/дедушки

  смеётся тот, кого больше нет»
- **Ключевики** (10-15 шт.):
  - оживить старое фото
  - старая фотография в видео
  - оживить фото бабушки
  - сделать видео из старой фотографии
  - вернуть к жизни фото
  - анимировать старое фото
  - ИИ оживляет фото
  - нейросеть оживляет фото
  - превратить фото в видео ИИ
  - оживить умершего родственника
- **UTM:** `?utm_source=yandex&utm_medium=cpc&utm_campaign=memory`
- **Минус-слова:** курсы, обучение, фотошоп, бесплатно, скачать
- **Бюджет/день:** 200₽ (тест 3-5 дней)

### 2. Кампания «Детство» (babies)

- **Креатив:** `videos/childhood.mp4` + копирайтинг для мам

  - оживить фото ребёнка
  - превратить детское фото в видео

  - оживить семейные фото
- **UTM:** `?utm_campaign=babies`
- **Аудитория:** интересы — Беременность, Воспитание детей, Семейные ценности

### 3. Кампания «Питомцы» (pets)

## Стратегия Директа

1. **Старт:** «Максимум кликов с ограничением CPC» — собираем первые 50-100
   конверсий в PHOTO_UPLOADED + GENERATION_STARTED. CPC лимит:
   - Поиск: 30₽ макс
   - РСЯ: 12₽ макс
2. **После 50+ конверсий:** переключение на «Максимум конверсий по цели»
   - **Цель:** `PAYMENT_SUCCESS` (ID см. в `.env` `YANDEX_GOAL_PAYMENT_SUCCESS`)
   - **CPA лимит:** 200₽ (если средний чек 290₽, остаётся маржа)
3. **После 200+ конверсий:** автостратегия + оплата за конверсии (CPA)

## Получение OAuth-токена (если идём по Пути A)

1. Зайти на https://oauth.yandex.ru/client/new
2. Создать приложение `botisk-admin`:
   - Платформа: **Веб-сервисы**
   - Callback URL: `https://oauth.yandex.ru/verification_code`
3. Доступы (поставить галочки):
   - ✅ Метрика → Управление целями, представлениями и счетчиками
   - ✅ Метрика → Чтение статистики
   - ✅ Директ → Использование Директ-API
4. Сохранить, получить **Client ID** (16 символов) и **Client Secret** (32 симв.)
5. Открыть в браузере: `https://oauth.yandex.ru/authorize?response_type=token&client_id=<CLIENT_ID>`
6. Подтвердить → редирект → в URL будет `#access_token=y0_AgAA...`
7. Скопировать токен в `.env`:
```
YANDEX_OAUTH_TOKEN=y0_AgAA...
```

## API-эндпоинты Yandex.Direct

Base URL: `https://api.direct.yandex.com/json/v5/`

Заголовки:
```
Authorization: Bearer y0_AgAA...
Accept-Language: ru
Client-Login: <логин в Я.Директе>   (если используется агентский аккаунт)
```

### Создать кампанию

```
POST /campaigns
{
  "method": "add",
  "params": {
    "Campaigns": [{
      "Name": "VideoAI — Память",
      "StartDate": "2026-05-22",
      "Type": "TEXT_CAMPAIGN",
      "ClientInfo": "vlz.transs",
      "DailyBudget": {"Amount": 20000000, "Mode": "STANDARD"},  // 200₽ в micros
      "TextCampaign": {
        "BiddingStrategy": {
          "Search": {"BiddingStrategyType": "HIGHEST_POSITION"},
          "Network": {"BiddingStrategyType": "MAXIMUM_COVERAGE"}
        },
        "CounterIds": {"Items": [109293181]},
        "Settings": [{"Option": "ADD_OPENSTAT_TAGS", "Value": "YES"}]
      }
    }]
  }
}
```

### Создать группу объявлений

```
POST /adgroups
{
  "method": "add",
  "params": {
    "AdGroups": [{
      "Name": "Память — основная",
      "CampaignId": <campaignId from previous step>,
      "RegionIds": [225],   // 225 = Россия
      "TextAdGroup": {}
    }]
  }
}
```

### Создать объявление

```
POST /ads
{
  "method": "add",
  "params": {
    "Ads": [{
      "AdGroupId": <adGroupId>,
      "TextAd": {
        "Title": "Оживите старое фото",
        "Title2": "ИИ за 60 секунд • Первое бесплатно",
        "Text": "Превратите фотографию близких в живое видео. Загрузите фото — получите видео за минуту. botisk.ru",
        "Href": "https://botisk.ru/?utm_source=yandex&utm_medium=cpc&utm_campaign=memory&utm_content={ad_id}",
        "DisplayUrlPath": "память"
      }
    }]
  }
}
```

### Добавить ключевые слова

```
POST /keywords
{
  "method": "add",
  "params": {
    "Keywords": [
      {"AdGroupId": <id>, "Keyword": "оживить старое фото", "Bid": 30000000},  // 30₽
      {"AdGroupId": <id>, "Keyword": "старая фотография в видео", "Bid": 25000000},
      ...
    ]
  }
}
```

### Минус-слова на уровне кампании

```
POST /campaigns
{
  "method": "set",
  "params": {
    "Campaigns": [{
      "Id": <campaignId>,
      "TextCampaign": {
        "Settings": [{
          "Option": "MAINTAIN_NETWORK_CPC",
          "Value": "YES"
        }],
        "NegativeKeywordSharedSetIds": {"Items": []}
      }
    }]
  }
}
```

## Что Claude должен спросить у пользователя ПЕРЕД запуском

1. **Логин в Директе** (нужен в Header `Client-Login`, если есть)
2. **Регионы помимо России** — нужно ли СНГ
3. **Время показа** — круглосуточно или по расписанию (например, исключить ночь)
4. **Утверждение текстов объявлений** — Claude сгенерит 2-3 варианта, пользователь выбирает
5. **Бюджет/день** — отличается ли от рекомендованных в этом доке

## Что Claude НЕ должен делать без подтверждения

- ❌ Запускать кампании в открутку (`State: ON` без явного OK)
- ❌ Менять бюджеты выше указанных
- ❌ Удалять/останавливать существующие кампании в этом аккаунте
- ❌ Пушить деньги на счёт Директа от имени пользователя

## Когда понадобится новый OAuth-токен

Токен Яндекса действует ~1 год. Если просрочен — `401 Unauthorized` от API.
Перевыпустить через те же шаги выше (новый visit на oauth.yandex.ru с тем же
client_id). Сменить значение в `.env`.
