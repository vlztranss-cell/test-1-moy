# Hello & Games

Набор простых Python-скриптов для практики.

## Скрипты

### hello.py
Спрашивает имя пользователя и выводит персонализированное приветствие.

```bash
python hello.py
```

### tictactoe.py
Крестики-нолики против компьютера с простым AI (блокирует, атакует, выбирает стратегические позиции).

```bash
python tictactoe.py
```

## Требования

Python 3.6+
